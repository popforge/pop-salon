# Index des changements (CHG)

Ce fichier est une index lisible des fichiers `CHG-*.yaml` présents dans ce dossier. Il peut être généré automatiquement par un script CI à partir des YAML.

## Usage

- Créez un fichier `CHG-YYYYMMDD-NN.yaml` à partir de `CHG-template.yaml`.
- Mettez à jour ce fichier si nécessaire (status, PR, commit).
