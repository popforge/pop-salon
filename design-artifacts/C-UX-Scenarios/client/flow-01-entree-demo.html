Démo — Flow 01 : Entrées multicanal

Cliquez sur un canal pour voir ce qu'il fait. Les canaux numériques redirigent vers la page de demande (Flow 02). Le canal "Appel téléphonique" simule l'interface coiffeuse.

Lien direct / QR
Exemple : URL unique partagée sur site ou QR code. Scannez le QR ou cliquez dessus pour ouvrir Flow 02 — Identification.
Facebook / Instagram
Clique sur le post → navigateur → page de demande.
Google Business Profile
Redirection vers la même URL de demande.
Messenger (bot)
Le bot renvoie un message contenant le lien de demande.
SMS (mot-clé)
Le système répond avec un lien vers la page de demande.
Appel téléphonique
Le flux téléphone est assisté par la coiffeuse (voir documentation).
Notes

La page de demande utilisée ici est : flow-02-identification.html

Utilisez cette démo pour vérifier que les canaux convergent vers la même URL et pour valider les textes et règles avant mise en production.
